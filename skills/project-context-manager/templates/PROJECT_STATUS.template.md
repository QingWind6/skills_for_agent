## 🔧 技术决策记录

### 决策 #001: UI 框架选择
- **日期**: 2026-03-10
- **决策**: 使用 Dear ImGui + 自定义复古主题
- **原因**: 用户希望实现经典 Macintosh 复古风格，ImGui 提供完全自定义能力
- **影响**: 需要自己实现所有 UI 组件，但获得完全的视觉控制

### 决策 #002: 窗口系统选择
- **日期**: 2026-03-10
- **决策**: 使用 GLFW（优先）或 SDL2
- **原因**: 轻量、跨平台、与 ImGui 集成良好
- **影响**: 需要手动处理文件对话框等系统集成

   - 每完成一个模块，更新本文件的完成状态

   - C++ 标准: C++17/20

4. **测试要求**:
   - 每个模块完成后立即测试
